# 🎓 StudFlow - Student Record Management System

**StudFlow** is a modern, web-based Student Management System built with a Java backend and a premium glassmorphic frontend. It is designed to replace outdated manual record-keeping with a secure, cloud-native solution.

## 🚀 Live Demo
**Check out the live application on Hugging Face:**
[https://huggingface.co/spaces/Pravardhan45/student-flow](https://huggingface.co/spaces/Pravardhan45/student-flow)

## 💎 Key Features
- **Modern Dashboard**: A sleek, responsive UI with real-time statistics (Total Students & Average Marks).
- **Cloud-Native Database**: Integrated with **TiDB Cloud** for secure, distributed data storage.
- **Full CRUD Operations**: Create, Read, Update, and Delete student records instantly.
- **Cloud Persistence**: Records are saved in the cloud and remain safe after application restarts.
- **Secure Architecture**: Credentials managed via environment variables to protect sensitive data.

## 🛠️ Tech Stack
- **Backend**: Core Java (SE) with an embedded HTTP server.
- **Database**: TiDB Cloud (Distributed SQL) via **JDBC**.
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), and JavaScript (ES6).
- **Deployment**: Dockerized and hosted on Hugging Face Spaces.

## 📂 Project Structure
- `Main.java`: The core server and API logic.
- `StudentDAO.java`: Database operations (JDBC).
- `Student.java`: Student data model.
- `index.html`: The modern UI.
- `Dockerfile`: Configuration for cloud deployment.
